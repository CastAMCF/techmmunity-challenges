Crie uma função que retorne a maior string de um array!
A função deve receber 1 parametro, um array de strings, e retornar a string com mais caracteres desse array.

Exemplos:
getBigger(["foo", "five", "ducks"]) -> "ducks"

getBigger(["javascript", "python", "php"]) -> "javascript"

getBigger(["a", "b", "c"]) -> a
