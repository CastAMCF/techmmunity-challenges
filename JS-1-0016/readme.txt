Caesar's Cipher!
Crie uma função que receba 2 parametro, uma string (a frase) e um number (a rotação), e retorne uma string convertendo cada letra da frase para uma letra seguinte do alfabeto. (Ex: Se a rotação for 3, a -> d, b -> e e y -> b).

Exemplos:
caesarCipher("Techmmunity is the best community!", 2) -> "Vgejoowpkva ku vjg dguv eqoowpkva!"

caesarCipher("Techmmunity is the best community!", 3) -> "Whfkppxqlwb lv wkh ehvw frppxqlwb!"

caesarCipher("Techmmunity is the best community!", 4) -> "Xiglqqyrmxc mw xli fiwx gsqqyrmxc!"

caesarCipher("Techmmunity is the best community!", 34) -> "Bmkpuucvqbg qa bpm jmab kwuucvqbg!"
