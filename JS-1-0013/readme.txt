Random Element!
Crie uma função que receba 1 parametro, um array de strings, e retorne um elemento aleatório desse array.

Exemplos:
getRandom(["foo", "bar", "seven", "walmart"]) -> "seven"

getRandom(["foo", "bar", "seven", "walmart"]) -> "foo"

getRandom(["foo", "bar", "seven", "walmart"]) -> "walmart"
